Why it exists

The problem this solves

Consumer 3D printers are affordable and incredibly versatile — but they're limited to a narrow range of plastics. Casting unlocks a much wider material palette: silicone, rubber, resin, and more. The catch? Combining these two processes involves a lot of trial, error, and information scattered across forums and YouTube videos.

Different filament types — PLA, ASA, PETG — interact differently with casting compounds. Surface roughness, porosity, chemical compatibility, and dimensional accuracy all vary. Without a reference, you're guessing.

Cast 3D closes that gap. We ran structured tests, collected community results, and turned everything into a searchable, filterable guide so you can make confident material decisions in minutes, not hours.

Features

What you can do with it

🗺️

Get a recommendation fast

The Guide page walks you through a short decision tree — pick your casting material and printer type and get a ranked compatibility recommendation in just a few clicks. No forum diving required.

⚖️

Compare tests side by side

The Compare page lets you drill into individual test results across mold filament types, casting compounds, and surface finishes — so you can find the exact data point that matches your setup.

📤

Share your own findings

Run a test we haven't covered? Upload your results via the Add Data page. Every contribution makes the guide more useful for the entire maker community.

Run it locally

Clone the repo and run with Python + Flask on Windows, macOS, or Linux.

python -m venv venv
.\venv\Scripts\activate
pip install Flask Flask-Scss Flask-SQLAlchemy
python app.py
make install
make run

Windows only: if step 2 fails with a policy error, run Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass in PowerShell first. Then click the http:// link printed in the terminal to open the site.
